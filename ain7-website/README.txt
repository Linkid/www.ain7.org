The AIn7 portal code
=====================

Overview of contents and licensing:

apache/
 - The apache configurations used by ain7.com - public domain
forum/
 - A heavily customized PunBB install - GNU GPL v2
   (All customizations are also GNU GPL v2 licensed)
 - Modified Easy Poll+ plugin
media/
 - Static content (css, images)
planet/
 - Planet configuration and templates = public domain
ain7/
 - Several homegrown django applications, GNU GPL v2
 hashcash/
  - Spam protection
 map/
  - Google maps
 news/
  - Site news
 pages/
  - General pages, with wiki syntax
 pastebin/
  - Pastebin with syntax highlighting and spam protection
 profile/
  - User profiles
 sources/
  - The infamous source-o-matic sources.list generator
 tags/
  - Template tages and views to generate planet/punbb templates
 templates/
  - The django templates
