A propos de ce site

Ce site est le site de l'AIn7. Il a été conçu bénévolement par des étudiants de l'ENSEEIHT et des anciens ingénieurs de l'ENSEEIHT qui se sont reproupés au sein d'une équipe. Si vous souhaitez contribuer (et nous vous y encourageons), n'hésitez pas à nous contacter et à rejoindre l'équipe.

Ce site est opensource est le code est disponible dans le dépôt bazaar de l'équipe associée au projet.

Nous vous encourageons à rapporter les bugs que vous rencontrez (sur Launchpad) ainsi qu'à contribuer au code !

Si vous souhaitez contribuer ou donner des idées, n'hésitez pas à contacter le webmaster.
